# Insomnium Security Policy

Insomnium is a fork of Insomnia, and is designed to be 100% local with 100% data privacy. By design, there should not be any security vulnerabilities in Insomnium.

If you find a security vulnerability in Insomnium, please report it privately using [GitHub's vulnerability reporting](https://github.com/andrelcunha/insomnium/security/advisories/new). Do not open a public issue.
